# jackdou-chinamap
laravel 中国地图web Api集合

## 功能介绍

  地图是大家经常在应用中需要使用到的功能，包括用户定位，Ip地址转换，地理位置正/逆编码等等。
  国内目前有很多地图API提供接口使用，但是每个公司的接口都不一样，为了方便使用不同的地图API，故写了这个包。
  目前集合了国内比较大的主流地图，包括百度，高德，腾讯等，实现了一些常用的功能，并且功能还会不断的扩展，
